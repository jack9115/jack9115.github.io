EmEditor专业版20.1.3

发表于 2020-10-01 Valine：

EmEditor Professional是功能强大的文本编辑器的专业版本，适用于程序员，Web开发人员和普通用户，具有语法突出显示和unicode支持。该编辑器具有以下功能：使用正则表达式搜索和替换文件，自动检测编码，突出显示链接和电子邮件地址，块选择模式，自定义按钮栏，菜单，字体和元素颜色。可以为每种语言创建单独的配置。

常规信息：

类型：安装，解压缩便携式x86 / x64
语言：俄语，英语，多语言
激活：ADMIN @ CRACK ，addhaloka，pawel97
另外：import EmEditor.reg

命令行参数：

静默安装：/ S / I
拆包：/ S / P
选择安装位置：/ D = PATH

/ D = PATH参数应指定为最新参数，
例如：EmEditor.v18.7.0.exe / S / I / D = C：\ MyProgram

新增功能： 官方网站
